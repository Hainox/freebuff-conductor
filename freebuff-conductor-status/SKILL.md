# freebuff-conductor-status

Обзор текущего состояния проекта и всех треков.

## When to use

- «На какой мы стадии?»
- «Что осталось сделать?»
- Перед началом нового трека
- Для статус-отчёта

## Instructions

### Step 1: Прочитай реестр

`conductor/tracks.md` → список всех треков.

### Step 2: Для каждого трека

Прочитай `metadata.json` + `plan.md`.

### Step 3: Сформируй отчёт

```
📊 freebuff Conductor — Статус проекта

## Активные треки
| ID | Описание | Тип | Статус | Прогресс |
|----|----------|-----|--------|----------|

## Завершённые треки
| ID | Описание | Тип | Задач |

## Детали активного трека: <track_id>
### Phase 1: ✅
- [x] Task 1.1
### Phase 2: ⚡ (in progress)
- [x] Task 2.1
- [~] Task 2.2 ← текущая
- [ ] Task 2.3

Всего: X/Y задач (Z%)
```

### Step 4: Рекомендации

- `in_progress` → «Продолжить?» (freebuff-conductor-implement)
- Все `completed` → «Новый трек?» (freebuff-conductor-new-track)
- `pending` → «Начать первый в очереди?»
